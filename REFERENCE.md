# REFERENCE

## Portugues (pt)

**Primeiro passo (importante):** altere o idioma no painel lateral, em `Idioma`.

### Funcoes principais

- Ajustar idioma da interface (`pt`, `en`, `es`, `nl`, `it`).
- Ajustar tema (`System`, `Light`, `Dark`).
- Abrir `Setup -> Config UI` para fonte/tamanho/densidade.
- Ver informacoes da versao em `Help -> About`.
- Encerrar o programa em `File -> Exit`.

### Menu

- `File -> Exit`: fecha o aplicativo.
- `Setup -> Config UI`: abre configuracoes visuais.
- `Help -> About`: abre dialogo de informacoes.

### Atalhos / Teclas

- `Alt`: foca a barra de menu.
- `Alt + F`, depois `X`: sair (dependendo do sistema/tema).
- `Tab` / `Shift+Tab`: navegar entre controles.
- `Enter`: confirmar acao em dialogos.
- `Esc`: fechar/cancelar dialogos (quando suportado pelo sistema).

## English (en)

**First step (important):** change the app language in the left panel, under `Language`.

### Main features

- Change UI language (`pt`, `en`, `es`, `nl`, `it`).
- Change theme (`System`, `Light`, `Dark`).
- Open `Setup -> Config UI` for font/size/density.
- Check build/version info in `Help -> About`.
- Exit the app in `File -> Exit`.

### Menu

- `File -> Exit`: closes the app.
- `Setup -> Config UI`: opens visual settings.
- `Help -> About`: opens information dialog.

### Shortcuts / Keys

- `Alt`: focus menu bar.
- `Alt + F`, then `X`: exit (depends on OS/theme).
- `Tab` / `Shift+Tab`: move across controls.
- `Enter`: confirm actions in dialogs.
- `Esc`: close/cancel dialogs (when supported by the system).
